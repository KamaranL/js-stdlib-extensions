## Installation and Usage

### Prerequisites

- [Node.js](https://nodejs.org/) `>=20.10.0`

### Install

```bash
npm i js-stdlib-extensions
```

### Usage

#### Import into package

```javascript
// javascript
require('js-stdlib-extensions')

// typescript
import 'js-stdlib-extensions'
```
